# Crypto Liquidation Insurance Frontend

A modern frontend application for crypto liquidation insurance for option prices.

## Features

- Clean, modern UI with glassmorphism design
- Price input for crypto options
- Insurance premium calculation
- Responsive design
- Real-time form validation

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

The application will be available at `http://localhost:5173`

### Build

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## Tech Stack

- React 18
- Vite
- Tailwind CSS
- Modern ES6+ JavaScript

## Usage

1. Enter the option price you want to insure
2. Fill in additional details (strike price, expiration date, etc.)
3. Enter collateral and insurance coverage amounts
4. Click "Calculate Insurance Premium" to see the estimated premium
